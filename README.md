This project presents a Centralized Healthcare Inventory & Prescription Management System designed to address common challenges in healthcare facilities such as expired medicine distribution, poor inventory control, prescription misuse, and inefficient record management. The system integrates patient records, staff management, medicine inventory, supplier information, prescriptions, and transaction logs into a unified database. By leveraging Oracle SQL, PL/SQL, and Oracle APEX, the application automates critical workflows including stock management, prescription dispensing, inventory auditing, and role-based access control. Database triggers prevent the issuance of expired medicines, procedures streamline prescription processing, and audit logs ensure accountability and transparency. The project showcases the implementation of advanced DBMS concepts including ER modeling, normalization, constraints, triggers, procedures, functions, cursors, and frontend integration, making it a comprehensive healthcare database management solution.
